Prework Study Guide

HTML

  • Structures the content of a web page using elements like paragraphs, headings, and links
  • Supports embedding multimedia elements like images, videos, and audio
  • Provides a foundation for web development, which is styled with CSS and made interactive with JavaScript

Css

  • Defines styles for HTML elements, including layout, colors, and fonts
  • Allows for responsive design with media queries to adapt content for different screen sizes
  • Can be extended with pre-processors like SASS and LESS for more efficient styling

Git

  • Tracks changes to files, enabling developers to manage and revert to previous versions
  • Facilitates collaboration through features like branches, merges, and pull requests
  • Allows decentralized development, with each contributor having a complete repository copy

JavaScript

  • Adds interactivity to websites, such as form validation and animations
  • Manipulates HTML and CSS dynamically using the Document Object Model
  • Often used with libraries and frameworks like React, Angular, and jQuery
